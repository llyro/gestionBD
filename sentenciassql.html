Sentencias de MYSQL


Sentencias para trabajar con tablas en MySQL
Estas son las sentencias básicas más comunes para trabajar con tablas en MySQL extraídas de un página web oficial:
CREATE TABLE
ALTER TABLE
DROP TABLE
TRUNCATE TABLE
RENAME TABLE

CREATE TABLE MySQL


CREATE TABLE es la sentencia que tenemos que usar para crear una tabla en la base de datos. Su sintaxis básica es la siguiente:
MySQL CREATE TABLE IF NOT EXISTS nombre_tabla
ALTER TABLE MySQL
Con ALTER TABLE podemos modificar la estructura y especificaciones de las tablas y sus columnas en una base de datos. Su sintaxis básica es la siguiente:
ALTER [ONLINE | OFFLINE] [IGNORE] TABLE nombre_tabla [alter_specification [, alter_specification] ...] DROP TABLE MySQL
Con DROP TABLE eliminamos una tabla de la base de datos. Su sintaxis básica es la siguiente:
DROP TABLE IF EXISTS nombre_tabla
TRUNCATE TABLE MySQL
La finalidad de TRUNCATE TABLE vaciamos todo el contenido de una tabla de la base de datos, la tabla siguen manteniendo las columnas y sus especificaciones. Su sintaxis básica es la siguiente:
TRUNCATE TABLE nombre_tabla;
RENAME TABLE MySQL
Con RENAME TABLE cambiamos el nombre de una tabla de la base de datos. Su sintaxis básica es la siguiente:
RENAME TABLE nombre_tabla TO nombre_nuevo_tabla
